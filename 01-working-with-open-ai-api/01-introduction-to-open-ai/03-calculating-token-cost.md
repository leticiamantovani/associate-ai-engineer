# 2. Calculating token cost

## How to calculate the cost?


1. Define price per token (input and output)

```python
input_price_per_token = 0.15 / 1.000.000 # $0.0000015 per 1M tokens
output_price_per_token = 0.6 / 1.000.000 # $0.000006 per 1M tokens
```


15 cents per million tokens -> when we divide by 1.000.000, we get the price per 

2. Get the input tokens tokens

- Input token is the number of tokens that are sent to the model
- we can get it from: **response.usage.prompt_tokens**

3. Get the output tokens

- Output token is the number of tokens that are generated by the model
- we can get it from: **max_completion_tokens**


4. Calculate the cost

- (input_price_per_token * input_tokens) + (output_price_per_token * output_tokens)

```python
cost = (input_price_per_token * input_tokens) + (output_price_per_token * output_tokens)
```

5. Print the cost

```python
print(f"The cost is: ${cost}")
```
