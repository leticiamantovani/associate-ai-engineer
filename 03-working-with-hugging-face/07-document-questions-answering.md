# Document Question Answering — AI Lesson

## What is Document Question Answering?

Document Question Answering (Document QA) involves generating answers to questions about the contents of a document or text passage. It requires two inputs:

| Input | Description | Example |
|---|---|---|
| **Document** | A text-based file | Research paper, contract, user manual, PDF |
| **Question** | A specific text query | "What is the total revenue of Q3?" |

The answer is generated by analyzing the document content and can be a **direct quote** or a **paraphrased response**.
- **THIS IS NOT RAG (Retrieval-Augmented Generation) YET**
---

## Use Cases

Document QA is widely used to automate data extraction and analysis across industries:

- **Legal** — identifies clauses in contracts, such as termination terms
- **Finance** — extracts key figures like revenue and expenses from reports
- **Customer support** — retrieves answers to common questions from manuals or FAQs

---

## Case Study: Automating HR Queries

Imagine you're an ML engineer at a large company. The HR team is overwhelmed with employee questions about holidays, notice periods, and other policies — all stored in a multi-page PDF called `US-Employee_Policy.pdf`.

**Goal:** build a system that retrieves specific answers from the document automatically, saving HR hours and streamlining employee communication.

---

## Step 1 — Extract Text from the PDF

Use the `pypdf` library to load the PDF and extract its text content:

```python
from pypdf import PdfReader

# Load the PDF file
reader = PdfReader("US-Employee_Policy.pdf")

# Extract text from all pages into a single string
document_text = ""
for page in reader.pages:
    document_text += page.extract_text()
```

> `.pages` returns all pages as a list. `.extract_text()` pulls the raw text from each page. The loop combines everything into one string ready for the model.

---

## Step 2 — Create the Q&A Pipeline

With the text extracted, set up a `question-answering` pipeline and pass your question along with the document as context:

```python
from transformers import pipeline

# Load a lightweight Q&A model
qa_pipeline = pipeline(
    task="question-answering",
    model="distilbert-base-cased-distilled-squad"
)

# Ask a question about the document
question = "How many volunteer days are employees allowed per year?"

result = qa_pipeline(
    question=question,
    context=document_text
)

print(result)
# {'answer': '1', 'score': 0.98, 'start': 412, 'end': 413}
```

> The pipeline returns the answer extracted from the document, a confidence `score`, and the character positions (`start`, `end`) where the answer was found in the context.

---

## Step 3 — Wrap into a Reusable Function

To make the system user-friendly, wrap the pipeline into a function that anyone can call:

```python
from pypdf import PdfReader
from transformers import pipeline

def load_document(pdf_path):
    reader = PdfReader(pdf_path)
    text = ""
    for page in reader.pages:
        text += page.extract_text()
    return text

def ask(question, document_text, qa_pipeline):
    result = qa_pipeline(question=question, context=document_text)
    return result["answer"]

# Setup
qa_pipeline = pipeline("question-answering", model="distilbert-base-cased-distilled-squad")
document_text = load_document("US-Employee_Policy.pdf")

# Ask anything
print(ask("What is the notice period?", document_text, qa_pipeline))
print(ask("How many vacation days do employees get?", document_text, qa_pipeline))
```

> Wrapping into functions lets users ask questions directly without worrying about the underlying implementation — clean, reusable, and production-friendly.

## Putting It All Together

```python
from pypdf import PdfReader
from transformers import pipeline

# 1. Extract text from PDF
reader = PdfReader("US-Employee_Policy.pdf")
document_text = ""
for page in reader.pages:
    document_text += page.extract_text()

# 2. Create Q&A pipeline
qa_pipeline = pipeline(
    task="question-answering",
    model="distilbert-base-cased-distilled-squad"
)

# 3. Ask a question
result = qa_pipeline(
    question="How many volunteer days are employees allowed per year?",
    context=document_text
)

print(result["answer"])  # 1
```

## Quick Reference

| Step | Tool | Purpose |
|---|---|---|
| Load PDF | `pypdf` → `PdfReader` | Read the file |
| Extract text | `.pages` + `.extract_text()` | Convert pages to a string |
| Build pipeline | `pipeline("question-answering")` | Load the Q&A model |
| Ask question | `question=` + `context=` | Query the document |
| Get answer | `result["answer"]` | Extract the response |
