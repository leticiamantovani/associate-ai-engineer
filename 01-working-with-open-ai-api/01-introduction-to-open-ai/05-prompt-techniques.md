# 5. Prompt techniques


## 1. Clear instructions first

Giving the AI clear instructions first, then providing the text to work on. This helps the AI understand exactly what you want it to do.

Example:

prompt = """
Update the following biography with the new name, pronouns, and job title.

Biography:
Name: Alex
Pronouns: they/them
Job: Software Engineer
"""

## The AI will then process this prompt to produce the updated biography.
This way, the AI knows to focus on editing the biography based on your instructions.
